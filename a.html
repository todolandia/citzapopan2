Sesión #3: Condiciones con IF

Aprendiendo a tomar decisiones en LUA

¿Qué es IF?

El comando if sirve para que el juego pueda tomar decisiones. El programa revisa si una condición se cumple. Si es verdadera, ejecuta una acción.

Ejemplo 1: IF en la consola

Este script revisa si el jugador tiene suficientes puntos.

local puntos = 10

if puntos >= 5 then
  print("¡Puedes pasar al siguiente nivel!")
end

Si la condición se cumple, aparecerá un mensaje en la Output Console.

Ejemplo 2: Bloque que cambia de color

Este script hará que un bloque cambie de color cuando el jugador lo toque.

script.Parent.Touched:Connect(function(hit)
  local humanoid = hit.Parent:FindFirstChild("Humanoid")

  if humanoid then
    script.Parent.Color = Color3.fromRGB(0,255,0)
  end
end)

El if revisa si lo que tocó el bloque es un jugador. Si lo es, el bloque cambia de color.

Actividad

  • Crea un nuevo Script en Roblox Studio.
  • Crea una variable llamada edad.
  • Usa un if para que, si la edad es mayor o igual a 10, aparezca el mensaje: "Puedes jugar este nivel".
  • Crea un bloque en el mapa.
  • Agrega un Script dentro del bloque.
  • Programa el bloque para que cambie de color cuando el jugador lo toque.
