Aprendiendo a programar en LUA

¿Qué es un Script?

Un script es un pequeño programa que le da instrucciones al juego sobre qué hacer. En Roblox, los scripts se escriben en el lenguaje LUA y sirven para crear acciones, reglas y comportamientos dentro de tu mundo.

Ejemplo: Bloque que mata al jugador

Vamos a crear un script sencillo para que, cuando el jugador toque un bloque, muera automáticamente.

-- Coloca este Script dentro del bloque
script.Parent.Touched:Connect(function(hit)
  local humanoid = hit.Parent:FindFirstChild("Humanoid")
  if humanoid then
    humanoid.Health = 0
  end
end)

Actividad 1

Crea un script que muestre un mensaje en la consola utilizando el comando print(). Ejemplo:

print("Hola, este es mi primer script en Roblox!")

Variables Locales

Una variable local es como una “cajita” donde guardamos información temporal. Por ejemplo:

local nombre = "Roblox"
print("Bienvenido a " .. nombre)

El operador .. sirve para unir texto con variables.

Condiciones con IF

El if nos permite tomar decisiones en el juego. Por ejemplo:

local puntos = 10
if puntos >= 10 then
  print("¡Felicidades, ganaste!")
end
