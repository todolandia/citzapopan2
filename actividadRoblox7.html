Sesión #7: Sistema de Monedas

Recolectar monedas dentro del juego

¿Qué es un sistema de monedas?

Un sistema de monedas permite que el jugador pueda ganar recompensas dentro del juego. Las monedas pueden servir para comprar objetos, desbloquear niveles o mejorar habilidades.

Para lograrlo usaremos leaderstats, que es un sistema de Roblox que muestra estadísticas del jugador en la tabla de puntuación.

Paso 1: Crear Leaderstats

Este script crea una tabla de monedas para cada jugador cuando entra al juego.

game.Players.PlayerAdded:Connect(function(player)

  local leaderstats = Instance.new("Folder")
  leaderstats.Name = "leaderstats"
  leaderstats.Parent = player

  local monedas = Instance.new("IntValue")
  monedas.Name = "Monedas"
  monedas.Value = 0
  monedas.Parent = leaderstats

end)

Paso 2: Crear una moneda en el mapa

Cuando el jugador toca la moneda, gana puntos.

script.Parent.Touched:Connect(function(hit)

  local player = game.Players:GetPlayerFromCharacter(hit.Parent)

  if player then
    local monedas = player.leaderstats.Monedas
    monedas.Value = monedas.Value + 10

    script.Parent:Destroy()
  end
end)

Paso 3: Duplicar monedas

Puedes copiar la moneda varias veces en el mapa para que los jugadores las recolecten mientras exploran el juego.

  • Crea una moneda con forma de cilindro o esfera.
  • Cambia su color a dorado.
  • Agrega el script anterior.
  • Duplica la moneda en diferentes lugares del mapa.
