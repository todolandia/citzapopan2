Sesión #6: Crear Tiendas

Tres formas de obtener objetos en Roblox

¿Qué es una tienda en Roblox?

Una tienda permite que el jugador pueda obtener objetos dentro del juego. Puede funcionar de diferentes maneras: tocando un objeto, haciendo clic o usando monedas.

Tienda 1: Tienda con Clic

El jugador obtiene una espada cuando hace clic en la tienda.

local espada = game.ReplicatedStorage:WaitForChild("Espada")

script.Parent.ClickDetector.MouseClick:Connect(function(player)
  local nuevaEspada = espada:Clone()
  nuevaEspada.Parent = player.Backpack
end)

Tienda 2: Tienda con Touch

El jugador recibe una espada cuando toca la tienda.

local espada = game.ReplicatedStorage:WaitForChild("Espada")

script.Parent.Touched:Connect(function(hit)
  local player = game.Players:GetPlayerFromCharacter(hit.Parent)

  if player then
    local nuevaEspada = espada:Clone()
    nuevaEspada.Parent = player.Backpack
  end
end)

Tienda 3: Tienda con Monedas

El jugador solo obtiene la espada si tiene suficientes monedas.

local precio = 50
local espada = game.ReplicatedStorage:WaitForChild("Espada")

script.Parent.Touched:Connect(function(hit)
  local player = game.Players:GetPlayerFromCharacter(hit.Parent)

  if player then
    local monedas = player.leaderstats.Monedas

    if monedas.Value >= precio then
      monedas.Value = monedas.Value - precio
      local nuevaEspada = espada:Clone()
      nuevaEspada.Parent = player.Backpack
    end
  end
end)
